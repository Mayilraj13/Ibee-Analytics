# 🚀 Installation & Setup

## Step 1 — Clone Repository

```bash
git clone https://github.com/Mayilraj13/Ibee-Analytics.git

cd Ibee-Analytics
```

---

# ⚙️ Backend Setup

## Step 1 — Navigate to Backend

```bash
cd backend
```

---

## Step 2 — Install Dependencies

```bash
npm install
```

Common packages:

```text
express
cors
dotenv
mongoose
axios
firebase-admin
```

---

## Step 3 — Configure Environment Variables

Create `.env`:

```env
PORT=5000
MONGO_URI=your_database_uri
API_KEY=your_api_key
FIREBASE_CONFIG=your_firebase_config
```

---

## Step 4 — Run Backend Server

```bash
npm start
```

or

```bash
node server.js
```

Expected output:

```bash
Server running on port 5000
Database connected successfully
```

---

# 💻 Frontend Setup

## Step 1 — Open New Terminal

```bash
cd frontend
```

---

## Step 2 — Install Dependencies

```bash
npm install
```

---

## Step 3 — Start Frontend

### React

```bash
npm start
```

### Vite

```bash
npm run dev
```

---

## ✅ Expected Output

```bash
http://localhost:3000
```

or

```bash
http://localhost:5173
```

# 🤖 AI Module Setup

## Step 1 — Navigate to AI Models

```bash
cd ai-models
```

---

## Step 2 — Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## Step 3 — Install AI Dependencies

```bash
pip install -r requirements.txt
```

Common libraries:

```text
pandas
numpy
scikit-learn
tensorflow
matplotlib
```

---

## Step 4 — Run AI Analytics

```bash
python predictive_analysis.py
```

Expected output:

```text
AI Analytics Started
Prediction Model Loaded
Dashboard Data Generated
```

# 🐛 Common Issues & Fixes

| Problem                      | Solution                            |
| ---------------------------- | ----------------------------------- |
| `npm install failed`         | Delete `node_modules` and reinstall |
| `Port already in use`        | Change server port                  |
| `Dashboard not loading`      | Verify backend API                  |
| `AI model not loading`       | Check Python dependencies           |
| `Database connection failed` | Verify `.env` credentials           |
| `CORS issue`                 | Enable CORS middleware              |
| `Charts not rendering`       | Verify chart libraries              |
| `Cloud deployment failed`    | Check deployment configuration      |
